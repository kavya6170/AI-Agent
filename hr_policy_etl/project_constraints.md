# Project Constraints & Scope
## Group 2: Data Ingestion & ETL Pipeline

### Scope Rules
1. **Text Extraction**: Digital documents (PDF, DOCX, TXT).
2. **OCR**: ONLY as fallback for scanned PDFs.
3. **Processing**: Text cleaning, semantic chunking, metadata attachment.
4. **Output**: Clean, chunked text with metadata.

### Hard Restrictions
- NO System Architecture design.
- NO Scope modification.
- NO ML, RAG, Agents, Vector DBs, or LLM calls.
- NO Embeddings.
- NO UI or APIs.

### Technical Requirements
- Language: Python.
- Style: Clean, modular, no unnecessary abstractions.
